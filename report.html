Website Report

Technical documentation and implementation details

Home Page
Page Markup Responsive Description
index.html HTML5, CSS3, W3CSS Yes Main landing page featuring hero video background with overlay, game introduction section with embedded YouTube video, latest updates section, and game features cards. Uses advanced CSS for video overlay effects, flexbox for centering, and responsive iframe containers. Complex z-index layering for nav bar positioning over video background. The game info section was particularly difficult as the W3CSS grid breakpoints are limited when it comes to 4 | 2 | 1 type layouts on larger screens. Custom CSS and media queries were used to adjust the breakpoints.
Heroes Page
Page Markup Responsive Description
heroes.html HTML5, CSS3, W3CSS Yes Interactive hero showcase. Features hover effects on hero cards, dynamic Javascript filtering and CSS transforms for smooth animations. Uses W3CSS Grid for responsive layout. Complex hover states with image scaling and overlay effects. This page was not inherently difficult to create as the responsive grid seemed to work well with the hero cards. This was mainly due to the fact that the content was all the same size. Adding more content gave issues with the breakpoints. Some custom css was used to achieve a responsive layout. I initially intended to have hero detail pages for each hero but due to time constraints that idea was ultimately scrapped. the card filter logic is relatively simple, it uses HTML data attributes to check each card's role and only displays the ones that match the selected filter. The logic is similar to the example from ZinoTrust Academy on YouTube. The custom scroll on the filter buttons was created using the documentation from W3Schools.
Gallery Page
Page Markup Responsive Description
gallery.html HTML5, CSS3, W3CSS Yes Image gallery created by implementing W3CSS Rows Grid system for masonry-style layout, JavaScript for modal interactions, and smooth transitions. Features lazy loading concepts and optimized image display with aspect ratio preservation. This was quite a simple page to create and works well on all screens. Plugins could be used to allow users to view each image in detail but again, due to time constraints. That feature is not implemented.
News Page
Page Markup Responsive Description
news.html HTML5, CSS3, W3CSS Yes News article layout with featured article section and grid-based article cards. Uses CSS gradients for card backgrounds, hover animations with transform effects, and responsive grid system. Implements card-based design with consistent spacing and typography hierarchy. This page did give some difficulty with the card layouts and custom CSS was used to change the default breakpoints of W3CSS.
Contact Page
Page Markup Responsive Description
contact.html HTML5, CSS3, W3CSS Yes Contact form with glassmorphism design using backdrop-filter and rgba backgrounds. Features form validation, custom styled inputs, and responsive layout. Uses advanced CSS for form styling, JavaScript for form handling, and media queries for mobile optimisation. The glassmorphism technique is similar to a backgroud used by Kevin Powell. A simple JavaScript function shows an alert and redirects the user back to the home page. This page was relatively simple to make, and due to the nature of how block level elements work, the left aligned layout was quite easy to get the way I wanted.

404 Page
Page Markup Responsive Description
404.html HTML5, CSS3, W3CSS Yes Custom 404 error page with Overwatch-themed messaging and animated floating icons. Features CSS keyframe animations, gradient backgrounds, and responsive design. Includes smooth hover effects on the return button and floating background elements for visual interest. This page was simple enough to create as all the content is centered.
